# Aqua Freecam – FiveM

Aqua Freecam is a FiveM script designed to help players capture high-quality screenshots and cinematic footage with ease.

## Features
- Smooth free camera movement
- Perfect for screenshots and cinematic videos
- Easy to use
- Optimized for FiveM servers

## Usage
Use Aqua Freecam to create professional-looking pictures, videos, and cinematic scenes in FiveM.

## Installation
1. Download the script.
2. Add it to your server's `resources` folder.
3. Add the resource to your `server.cfg`.
4. Restart your server.

Enjoy creating amazing content with Aqua Freecam!
